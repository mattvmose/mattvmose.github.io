# mattvmose.github.io

### Introduction

This website serves as a hub for information regarding all 5 of the official North American Formula 1 races for the 2026 season. 

### Purpose

The purpose of this website i to provide a structured and engaging platform for sharing information about the 5 North American F1 Grand Prix races for the 2026 season. It is intended to provide users with a centralized place to learn more about these events, view ticket prices, and view the area surrounding the tracks.

### Research and Content Development

The information presented across the site is drawn from a variety of reputable sources:

- Official Formula 1 resources for track statistics and event pricing
- General web development references such as W3Schools for best practices in HTML, CSS and JavaScript
- Open source images through sites such as Unsplash.com

All content is curated to be accurate and appropriate for a general audience. However, the content displayed on this site is only applicable to the 2026 Formula 1 season.

### Target Audience

This website is primarily designed for F1 fans, as it exclusively contains information regarding F1 races in North America. Also, there are certain pieces of information on each page that are tailored for fans looking to attend any of these races, such as the ticket prices and the embedded map.

### Content Overview

The website is organized into two major sections across 6 pages:

- **Home Page**
    Provides an overview of the site and links to all major subpages.

- **Formula 1 Grand Prix pages**
    Individual pages covering:
    - United States Grand Prix
    - Miami Grand Prix
    - Las Vegas Grand Prix
    - Canadian Grand Prix
    - Mexico City Grand Prix
    Each page includes race details, track information, relevant images, ticket prices, and an embedded map of the track.

### Technical Overview

This site is built using:

- **HTML** for basic page structure
- **CSS** for layout, styling, and responsive elements
- **JavaScript** for small interactive features
- **GitHub Pages** for hosting and automatic deployment
- **Git** for version control and updates

Al pages follow modern web standards, including:

- Proper heading hierarchy
- Alt text for images
- Consistent navigation
- Accessible color contrast and layout
